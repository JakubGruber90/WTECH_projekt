Návrh databázy

Databáza bude pracovať v jazyku PostgreSQL a bude obsahovať celkovo 14 tabuliek, z toho 4 hlavné:

1. customers - obsahuje informácie o zákazníkoch - id zákazníka, krstné meno, priezvisko, heslo, email, telefónne číslo, adresa, mesto, štát 
2. products - obsahuje informácie o produktoch - id produktu, kategória, popis, cena, fotky produktu, pole dostupných veľkostí, materiál, značka produktu, pole dostupných farieb a počet produktov na sklade
3. orders - obsahuje všetky objednávky - id objednávky, id zákazníka, id produktov v objednávke, celková cena, id typu doručenia, id typu platby a id statusu objednávky
4. cart - obsahuje informácie o produktoch v košíku - id košíku, id produktov v košíku, cena a id zákaznika, ktorému košík patrí

Ďalej sú to 3 tabulky, ktoré upresňujú informácie o objednávkach:

5. payment_methods - obsahuje dostupné typy platby - id typu platby, metóda platby
6. shipping_types - obsahuje dostupné typy doručenia - id typu doručenia, názov typu doručenia a cena konkrétneho typu
7. order_statuses - obsahuje informácie o stavoch objednávky - id stavu a názov stavu

Ďalej sú to 3 tabuľky, ktoré dopĺňajú informácie o produktoch:
8. materials - obsahuje dostupné materiály - id materiálu a samotný materiál
9. colors - obsahuje dostupné farby - id farby a hexadecimálny tvar farby
10. sizes - obsahuje dostupné veľkosti - id veľkosti a veľkosť topánok v EU formáte

A nakoniec sú to 4 tabuľky, ktoré definujú vzťahy medzi ostatnými tabuľkami:
11. products_materials - spája tabuľky products a materials, určuje, aké materiály má konkrétny produkt
12. products_sizes - spája tabuľky products a sizes, určuje, aké veľkosti sú dostupné pre daný produkt
13. products_colors - spája tabuľky products a colors, určuje, aké farbý sú dostupné pre daný produkt
14. products_cart - spája tabuľky products a cart, určuje, aké konkrétne produkty sú v nákupnom košíku
